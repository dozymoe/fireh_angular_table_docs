FhInfiniteScroll

Good old infinite scroll, one direction, down only, will load all items in html, not recommended for dangerously large number of items.

Not compatible with FhTablePager.

At the moment we are using jQuery Infinite Scroll Helper

If none of the closest parent html element has style overflow-y:scroll, the infinite-scroll will trigger as you go down the whole page.

See fhInfiniteScroll demo
