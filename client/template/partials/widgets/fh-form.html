FhForm

This widget doesn't provide any html markup.

Helps setting up draft and form-data, and related events.

Angular Directive Parameters

  • data-fh-form itself

    Expected value, instance of FhTableDefinition.

  • data-fhp-table

    Expected value, instance of FhTableDefinition, if you haven't set one already with data-fh-form.

  • data-fhp-form-item

    Expected value, object, the form's data.

  • data-fhp-element-id

    Expected value, string, form elements usually have id attribute set, for example to link <label> with the element itself.

    If you don't set one, a random element-id will be assigned.

  • data-fhp-element-name

    Expected value, string, the alphabets part of random element-id, the priority is lower than value set in data-fhp-element-id.

    Random numbers will be appended when the id was constructed, making the id unique.

  • data-fhp-middlewares

    Expected value, comma separated list of services names.

  • data-fhe-*

    Custom event handler if you wanted to change this widget behaviour.
