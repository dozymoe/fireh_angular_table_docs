FhTablePager

Pager navigation.

Angular Directive Parameters

  • data-fh-table-pager itself

    Expected value, instance of FhTableDefinition.

  • data-fhp-table

    Expected value, instance of FhTableDefinition, if you haven't set one already with data-fh-table-pager.

  • data-fhp-template-url

    Expected value, string of url to partial html, with this you can override the template html used by the widget.

    The widget uses Angular $templateRequest on the value.

  • data-fhp-pager-links-count

    Expected value, integer, the total number of page offsets to render.

  • data-fhp-middlewares

    Expected value, comma separated list of services names.

  • data-fhe-*

    Custom event handler if you wanted to change this widget behaviour.
