FhTableBatchActions

Perform actions on selected rows.

Angular Directive Parameters

  • data-fh-table-batch-actions itself

    Expected value, instance of FhTableDefinition.

  • data-fhp-table

    Expected value, instance of FhTableDefinition, if you haven't set one already with data-fh-table-batch-actions.

  • data-fhp-middlewares

    Expected value, comma separated list of services names.

  • data-fhe-*

    Custom event handler if you wanted to change this widget behaviour.
