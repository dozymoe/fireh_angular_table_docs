FhTable

This widget doesn't provide any html markup.

A wrapper to FhTableListResourceControllerMixin, provides scope functions related to selection of all rows.

Angular Directive Parameters

  • data-fh-table itself

    Expected value, instance of FhTableDefinition.

  • data-fhp-table

    Expected value, instance of FhTableDefinition, if you haven't set one already with data-fh-table.

  • data-fhp-middlewares

    Expected value, comma separated list of services names.

  • data-fhe-*

    Custom event handler if you wanted to change this widget behaviour.

Demo

See FhTable demo.

The example shows how to create instance of FhTableDefinition in angular controller and use data-fh-table in html.

FhTable is the one responsible for getting data/items from the REST API.

Widgets Requirement

None.
