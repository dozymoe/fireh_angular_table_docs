Widgets

All packaged widgets are set as attribute directives.

The widgets don't have to be laid in an hierarchical structure. Your <table> can be located in the middle of the page while your filter and sorting widget in the sidebar or somewhere else.

Table related widgets

  • FhTable

    Contrary to what the name implies, you don't need to attach this widget to a <table>, and it doesn't provide any html markup.

    A wrapper to FhTableListResourceControllerMixin, provides scope functions related to selection of all rows.

    See fhTable demo.

  • FhTablePager

    Pager navigation.

    See fhTablePager demo

  • FhTablePagerSize

    Select pager sizes.

    See fhTablePagerSize demo

  • FhTableSorting

    See fhTableSorting demo

  • FhTableFilterText

    Filter table rows by field value.

    See fhTableFilterText demo

  • FhTableRow

    Allows you to interact with the data of each row, be it editing, deletion or other custom activities.

    See fhTableRow demo

  • FhTableFilterSelect

    Filter table rows by multiple selection.

    See fhTableFilterSelect demo

  • FhTableBatchActions

    Perform actions on selected rows.

    See fhTableBatchActions demo
