draftUnsetField

This is an action event, it is used to prompt other widgets to perform actions, if they succeded passive form of the event's name should be triggered to notify this widget that it was a success or a failure.

Modify the form draft, used by form input widgets to change their parent form's data.

Parameters

  1. event

    Angular event object.

  2. item
  3. fieldName
  4. value
  5. options

    For now it contains information about the triggering widget.

    Possible properties:

    • formId
