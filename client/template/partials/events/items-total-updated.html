itemsTotalUpdated

This is a display event, it is used to prompt other widgets to perform ui changes, the listeners should be passive (not performing any action).

Notification triggered after successful fetch of items from server, the response must contain the property: total, the number of all items, even those outside the current pager page.

Parameters

  1. event

    Angular event object.

  2. totalItems

    The number of all the items available on the server under current filter criteria.

  3. totalShelved

    The number of items fetched from the server so far under current filter criteria.
