addEditableField

This is an action event, it is used to prompt other widgets to perform actions, if they succeded then passive form of the event's name should be triggered to notify this widget that it was a success or a failure.

Widgets that implemented draft, FhForm and FhTableRow (in editable mode), white-listed the draft fields that can be submitted.

Allow form input widgets to set editable fields of the form's draft.

Parameters

  1. event

    Angular event object.

  2. fieldName
  3. options

    For now it contains information about the triggering widget.

    Possible properties:

    • formId
