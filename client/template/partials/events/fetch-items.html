fetchItems

This is an action event, it is used to prompt other widgets to perform actions, if they succeded passive form of the event's name should be triggered to notify this widget that it was a success or a failure.

Refresh the items, usually related to changes in the pager, order-by fields, or filter-by fields.

Parameters

  1. event

    Angular event object.

  2. options

    This is different from the usual form-widget related options.

    Possible properties:

    • flush (possible values: true, false)
    • initialFetchItems (possible values: true, false)
    • page (possible values: 'next', 'prev', number)
