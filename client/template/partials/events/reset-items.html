resetItems

This is an action event, it is used to prompt other widgets to perform actions, if they succeded then passive form of the event's name should be triggered to notify this widget that it was a success or a failure.

Clear items, and refetch from server with the pager offset restarted to page 1, usually happens because the filter-by or the order-by fields changed.

Parameters

  1. event

    Angular event object.
