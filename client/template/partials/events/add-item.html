addItem

This is an action event, it is used to prompt other widgets to perform actions, if they succeded passive form of the event's name should be triggered to notify this widget that it was a success or a failure.

Create user object. This event was triggered when form-user saved a form with the intention of creating new objects.

You are responsible of implementing the event listener, perform the necessary ajax call to persist the object, then trigger itemAdded if successful or itemAddFailed if failed.

If you used ajax, you should trigger ajaxRequestStarted and ajaxRequestFinished.

Parameters

  1. event

    Angular event object.

  2. draft

    The submission data or the form's current data.

  3. item

    The form's initial/original data.

  4. options

    For now it contains information about the triggering widget.

    Possible properties:

    • formId
