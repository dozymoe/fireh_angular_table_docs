FhSelectedItemsMixin

Help develop your own widget, provides consistent/uniform interface.

Store a read-only clone of selected items in the $scope.

Parameters

  1. scope

    Angular controller's $scope.

  2. options

    Object with the following properties: multipleSelection.

  3. cleanupList (optional)

    List of cleanup callbacks, such as one produced by $scope.$on and $attrs.$observe.
    The list will be modified inplace with cleanup callbacks generated by this mixin.

Results

  1. Return object with the following properties: eventCleanupCallbacks.
  2. Modify the third argument (cleanupList) inplace, the added callback functions will be the same as the ones in the returned object.
    The third argument is added to the api for simplicity.
  3. Modify $scope, variables created: $scope.data.selectedItems.
  4. Install event listeners.
