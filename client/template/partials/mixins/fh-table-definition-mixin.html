FhTableDefinitionMixin

Help develop your own widget, provides consistent/uniform interface.

Let you use html attributes data-fhp-table, which value is instance of FhTableDefinition.

It is possible to use custom html attribute instead of data-fhp-table.

Parameters

  1. scope

    Angular controller's $scope.

  2. attrs

    Angular controller's $attrs.

  3. alternateAttrField (optional)

    String, alternative html attribute name, the default is data-fhp-table.

Results

  1. Modify scope, create variable $scope.fhtable.
